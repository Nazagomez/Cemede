# Cemede

Sistema web para la estimación de la capacidad de carga turística y su impacto ambiental en las playas Junquillal de la Cruz y Playa Grande, Guanacaste.

**Programa:** CEMEDE-UNA  
**Stack:** React · FastAPI · MySQL · scikit-learn
